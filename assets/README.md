# Photo Placeholder

Place your profile photo here as `profile.jpg`

The portfolio website will automatically use this photo for the profile image.

## Recommended Specifications:
- Format: JPG, PNG, or WebP
- Size: 500x500 pixels (square)
- File size: Under 500KB for best performance

## How to Add Your Photo:

### Option 1: Direct Upload (Easiest)
1. Open `index.html` in your browser
2. Hover over the profile image
3. Click the upload icon that appears
4. Select your photo
5. Done! The photo is saved in your browser

### Option 2: Manual Placement
1. Save your photo as `profile.jpg` in this folder
2. The website will automatically display it

If no photo is provided, you can upload one directly through the website interface.
